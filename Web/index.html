TYPO3 CMS Composer Repository

What Is It?

This is a composer repository, enabling you to install TYPO3 CMS core and extensions including dependencies. We think it's much nicer to build up your deployment on composer, rather than git submodules.

This repository includes:

  • TYPO3 CMS core
    All versions currently supported by typo3.org
  • TYPO3 CMS extensions
    All Extensions, not marked as unsecure, in the TER
  • Composer Installer
    To install everyhing into it's right place (typo3conf/ext and typo3_src)

You can also list all available packages.

How to use

Using composer to install TYPO3 CMS is really simple. You just need to include this repository into your composer.json like this.

After you have done this, you can just require the core and / or extensions with the usual composer versioning format.

This will allways get you the most current 6.1.x core of TYPO3 CMS, the most current version of a1_ttnews (including it's dependency tt_news!).

That's it! Really ... just download the demo, run composer.json and be amazed.

How it works

Four times a day we crawl the TYPO3 Extension Repository and the available core downloads. We process all ext_emconf.php's (of every non-unsecure version) to calculate the dependencies. Thus making sure you allways get matching version numbers.

We generate the metadata (packages.json), needed by composer to find the right downloads. We don't host anything ourself, but only link to the right resources. Everything downloads directly from typo3.org, so you can be sure, to only get the best stuff.
